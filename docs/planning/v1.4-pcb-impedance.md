# v1.4 — PCB 阻抗计算器（pcb-impedance）planning

路由：`/tools/hardware/pcb-impedance`

---

## 1. 已知事实

- 项目已经有 `rc-calculator`（电阻分压 + RC 滤波 + 波特图），布局/Recharts 引用模式可参考。
- 项目无 SVG 重组件，现有工具的截面图用 inline SVG（如 resistor-calculator）。
- 单位：嵌入式 PCB 行业默认 mil（密耳）和 mm 混用，需要双单位支持。

## 2. 假设

- A1：**只用 IPC-2141 近似公式**，不上 2D 场求解器（如 wcalc / atlc）。这是有意识地选简单。
- A2：误差 ±5% 是"够用"的。**ToolIntro 必须明示**："近似公式，误差 ±5%，精确设计请用 Polar SI9000 / Hyperlynx"。
- A3：3 种几何就够：**微带线（Microstrip）/ 带状线（Stripline）/ 差分对（Differential pair）**。不做共面波导（CPW）、不做嵌入式微带（embedded microstrip）。
- A4：参数范围在常见 PCB 工艺内（W: 3-50 mil, T: 0.5-3 mil, H: 2-30 mil, εr: 2.0-6.0）。超出范围给 warning 但仍计算。
- A5：反查"给定 Z₀ 求 W"用牛顿迭代 + 二分兜底，最多 30 步，收敛阈值 0.01Ω。

## 3. 开放问题

### Q1 — 公式来源？

| 几何 | 候选公式 | 备注 |
|------|---------|------|
| Microstrip | IPC-2141 表 4-1 | 经典，与 SI9000 误差 < 5% |
| Stripline | IPC-2141 表 4-2 | 经典 |
| Differential | Wadell `Transmission Line Design Handbook` 12.2 | IPC-2141 没明确给差分公式 |

→ **倾向**：微带 + 带状用 IPC-2141；差分对用 Wadell 近似。**Source 在 lib 顶部注释明确标出**。请确认是否接受这种"杂交"。

### Q2 — UI 展示形式？

| 选项 | 说明 |
|------|------|
| (a) 单页三 tab：Microstrip / Stripline / Diff，每 tab 独立表单 | 现有工具风格一致 |
| (b) 单表单 + geometry 下拉切换 | 共享更多代码，但 UI 不直观 |

→ **倾向 (a)**：与现有 baudrate-calculator / data-structure 的 tab 模式一致。请确认。

### Q3 — SVG 截面图必要性？

候选：(a) 用 SVG 画截面（3 个几何各画一张，参数变化时按比例缩放）；(b) 仅用 CSS border + flex 模拟（更简单但不够"专业"）；(c) 不画图，纯数字。

→ **倾向 (a) SVG**：截面图是阻抗工具的"可信感"来源——用户没有图就不敢信结果。SVG 单 tab 一张，约 30 行，不复杂。请确认。

### Q4 — 反查方向（求 W 给 Z₀）值得做吗？

→ 这是工程师最常用的需求（"我要 50Ω，W 多少？"）。**值得做**。但要把"反查不收敛"的边界处理好：迭代失败时回退到正向扫描（W=3..50 mil 步进 0.5）+ 取最接近的。

### Q5 — 单位制？

→ **同时支持 mil 和 mm**，单选切换。所有内部计算用 mm（IPC-2141 公式以 mm 为单位）。**绝不**做 inch/cm 等乱七八糟单位。

## 4. 简化方案考察

### S1：能不能用纯 CSS border 画截面图？

→ 不行。差分对要画两条线 + 间距标注，CSS 做不出。SVG 是最简单选项。

### S2：能不能只做 microstrip 一种？

→ 量级问题。3 种几何的公式和 UI 重复度 70%。多做两种成本不高，省了用户去别处算的时间。**保留 3 种**。

### S3：能不能不做反查？

→ 不行。反查是工具的核心价值。正向计算用户用 Excel 也能做，反查才是工具的"卖点"。

## 5. 可验证成功标准

| # | 测试 | 参考值来源 |
|---|------|----------|
| 1 | Microstrip W=10mil T=1.4mil H=5mil εr=4.5 → Z₀ ≈ 50.3Ω ±3% | Polar SI9000 计算结果 |
| 2 | Stripline W=8mil T=1.4mil H=10mil εr=4.5 → Z₀ ≈ 50Ω ±3% | 同上 |
| 3 | Differential W=5 S=5 T=1.4 H=5 εr=4.5 → Z_diff ≈ 100Ω ±5% | 同上 |
| 4 | 反查：50Ω microstrip + H=5 T=1.4 εr=4.5 → W ≈ 10mil（迭代 < 20 步） | 反向求解 |
| 5 | 反查不收敛时回退扫描，30 步内必有结果 | 边界 |
| 6 | mil ↔ mm 互转：1 mil = 0.0254 mm，往返不丢精度 | 单位换算 |
| 7 | εr 超出 [2, 6] 给 warning 但不阻塞 | 警告 |
| 8 | W < 3 mil 给 warning（PCB 工艺最小线宽） | 警告 |
| 9 | 计算函数纯函数，相同输入相同输出 | 副作用 |
| 10 | 输出包含 ε_eff（有效介电常数）和传播延迟（ps/inch） | 完整性 |

10 个 vitest。
